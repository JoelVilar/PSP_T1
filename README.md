PRÁCTICA 1ER TRIMESTRE: INGRESO DE DATOS POR HILOS.
/*Ejecución en Windows.*/.
1.Abra la consola.
2.Escriba el comando "java -jar" seguido de la ruta absoluta del archivo. Ejemplo:java -jar C:\USER\PSP_T1\PSP_T1_InsertByThread.jar
3.Siga los pasos que se le indican. Introduzca las opciones con los números del teclado.
/*Ejecución en Linux*/
1.Abra la terminal.
2.Escriba el comando  "java -jar" seguido de la ruta absoluta del archivo. Ejemplo:java -jar /home/user/PSP_T1/PSP_T1_InsertByThread.jar
3.Siga los pasos que se le indican. Introduzca las opciones con los números del teclado.
/* Nota: En caso de no tener la jre instalada, deberá hacerlo previamente, por lo que la terminal le dirá los pasos a seguir.
 * Para ello, introduzca el comando "sudo apt install" seguido de la versión que desea. Ejemplo: "sudo apt install default-jre"(Utilice
 * una versión reciente a ser posible). Una vez finalizado, proceda a repetir el paso 2.*/
